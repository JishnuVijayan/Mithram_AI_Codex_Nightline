# AGENTS.md

## Goal
Build production-quality code while optimizing for hackathon delivery.

## Core Rules
- Commit frequently. Prefer small, logical commits instead of one large commit.
- Never hardcode secrets.
- Use service/repository architecture.
- Keep functions small.
- Ask before destructive schema changes.

## Product Rules
- Children can configure personalized greetings.
- Children can configure the questions asked during calls.
- Every month, start the call by asking for a changing security code to prove the call is genuine and not spam.
- Support retry, SMS, email, push notifications and emergency escalation.
- Respect privacy consent.
